
# Waveform player 


## Usage :

```
usage: python-waveform_player.py [-h] [-i INPUT] [-style STYLE] [-color]

optional arguments:
  -h, --help    show this help message and exit
  -i INPUT      INPUT FILE
  -style STYLE  Style : bar, split, blend, full
  -color        Show color
```
